# Markdown预览 浏览器插件

一个简单实用的浏览器插件，可以实时将Markdown文本转换为HTML预览。

## 功能特点

- 实时预览：输入Markdown文本时即时转换为HTML显示
- 简洁界面：清晰的双栏设计，左侧编辑，右侧预览
- 样式美观：预览区域采用现代化的样式设计，支持多种Markdown元素的美化
- 轻量级：无需额外依赖，安装即可使用

## 安装方法

1. 下载本项目的所有文件
2. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目所在的文件夹

## 使用说明

1. 点击浏览器工具栏中的插件图标
2. 在左侧输入框中输入Markdown格式的文本
3. 右侧区域会实时显示转换后的HTML预览效果

## 支持的Markdown语法

- 标题（H1-H6）
- 粗体、斜体
- 有序、无序列表
- 链接和图片
- 引用块
- 代码块
- 水平分割线
- 表格

## 技术实现

- 使用Chrome Extension Manifest V3
- 采用marked.js进行Markdown转换
- 自定义CSS样式美化预览效果

## 注意事项

- 本插件仅支持基本的Markdown语法
- 图片预览需要正确的图片URL
- 建议定期保存重要内容，避免意外丢失
